# VirtualsWatch

A real-time monitoring tool for newly launched Virtuals agents.

## Features

* Tracks newly listed agents during their first 2 hours
* Monitors holder growth and activity signals
* Detects unusual early traction
* Sends instant alerts via Telegram and Feishu
* Helps users discover promising projects faster

## Purpose

With official pre-announcements no longer available, VirtualsWatch provides automated monitoring and notification to reduce manual tracking and improve response speed.
